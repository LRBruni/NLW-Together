# Beautysalon

Esse foi o projeto da NLW Together de 2021 da Rocketseat.
O projeto consiste em uma lading-page de um salão de beleza feito em HTML, CSS e um pouco de Javascript.
A página é completamente adaptável para diferentes plataformas.
